## Lab One
Python script to create a **database**


## Run Locally

Clone the project

```bash
  git clone https://github.com/PLP-Database-DEPT/lab-1.git
```

Go to the project directory

```bash
  cd lab-1
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Create Database

```bash
  python create_db.py
```
